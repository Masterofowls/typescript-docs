## What You'll Learn

This documentation takes you from absolute zero to writing professional, type-safe TypeScript applications. Each section builds on the last, with real code you can run today.

| Level | Topics |
|-------|--------|
| 🟢 **Beginner** | Setup, `index.ts`, primitive types, interfaces, basic classes |
| 🟡 **Intermediate** | Generics, modules, decorators, utility types |
| 🔴 **Advanced** | Conditional types, mapped types, design patterns, DI |
| 🚀 **Pro** | Express APIs, REST routes, middleware, Jest testing |
