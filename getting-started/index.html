linkGetting Started with TypeScript

TypeScript is a strongly typed superset of JavaScript maintained by Microsoft. It compiles to plain JavaScript and runs anywhere JavaScript runs.

linkWhy TypeScript?

1link// ❌ JavaScript — no safety

2linkfunction add(a, b) {

3link  return a + b;

4link}

5linkadd(1, "2"); // Returns "12" silently!

6link

7link// ✅ TypeScript — caught at compile time

8linkfunction add(a: number, b: number): number {

9link  return a + b;

10link}

11linkadd(1, "2"); // Error: Argument of type 'string' is not assignable to parameter of type 'number'

TypeScript gives you:

  • Static type checking — catch bugs before runtime
  • IntelliSense — rich auto-complete in your editor
  • Refactoring safety — rename symbols across your whole project
  • Self-documenting code — types serve as inline documentation

linkInstallation

1link$# Install TypeScript globally

2link$npm install -g typescript

3link$

4link$# Check version

5link$tsc --version

For a project:

1link$npm init -y

2link$npm install --save-dev typescript @types/node ts-node

linkYour First TypeScript Project

1link$mkdir my-ts-app

2link$cd my-ts-app

3link$npm init -y

4link$npm install --save-dev typescript ts-node @types/node

5link$npx tsc --init

This creates a tsconfig.json configuration file.

linkRunning TypeScript

1link$# Compile to JavaScript

2link$tsc index.ts

3link$

4link$# Run directly (development)

5link$ts-node index.ts

6link$

7link$# Or using npx

8link$npx ts-node index.ts
